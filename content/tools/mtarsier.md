# mTarsier - 开源 MCP 服务器管理平台

## 🎯 解决什么痛点？
MCP（Model Context Protocol）服务器越来越多，但管理混乱：配置复杂、版本冲突、客户端不兼容...mTarsier 提供统一的管理平台，让 MCP 服务器的部署、配置、监控变得像管理手机 App 一样简单！

- **开源爱好者**：喜欢折腾最新 AI 协议的极客

- **统一管理**：一个平台管理所有 MCP 服务器和客户端
- **开源免费**：完全开源，可自建部署，数据自主可控
